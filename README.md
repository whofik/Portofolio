# Muhammad Fikri - Portfolio

Portfolio website built with Vite + React.

![Portfolio Preview](https://img2.pixhost.to/images/6706/708113572_image.jpg)

## Features

- Responsive Design (Mobile, Tablet, Desktop)
- Smooth Animations
- Modern UI/UX
- SEO Optimized
- Fast Performance

## Sections

- About
- Skills
- Certificates
- Music
- Contact

## Tech Stack

- React 19
- Vite
- CSS3
- Inter Font

## Installation

```bash
git clone https://github.com/whofik/Portofolio.git
cd Portofolio
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

## Deployment

Deploy on [Vercel](https://vercel.com)
